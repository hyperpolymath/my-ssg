# Welcome to NoteG

This is an example site built with the NoteG Static Site Generator.

## Features

- **Fast** - Built on Deno runtime
- **Type-safe** - Ada/SPARK verified core
- **Accessible** - BSL, GSL, ASL, Makaton support
- **Flexible** - 28 SSG adapters via MCP

## Getting Started

1. Create your content in the `content/` directory
2. Customise templates in the `templates/` directory
3. Run `just serve` to preview
4. Run `just build` to generate your site
